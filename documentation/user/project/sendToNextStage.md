# Create Project

Move project to next stage

**URL** : `/projects/sendToNextStage`

**Method** : `POST`

**Auth required** : Yes

**Permissions required** : None

**Parameters** : projectId, `[`nextStage`]`

## Success Response

**Code** : `200 OK`

**Content examples**


```json
{
    "success": true,
    "msg": "Project moved to next stage"
}
```
