# Show Approved Users

List all the users approved by the admin

**URL** : `/approvedUsers`

**Method** : `GET`

**Auth required** : Yes

**Permissions required** : None

**Parameters** : 	projectName, division,landUser,lotNo


## Success Response

**Code** : `200 OK`

**Content examples**


```json
{
    {
    "success": true,
    "msg": "created database entry"
    }
}
```
