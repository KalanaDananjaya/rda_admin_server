
# Show Rejected Users

List all the users rejected by the admin

**URL** : `/user/rejectedUsers`

**Method** : `GET`

**Auth required** : Yes

**Permissions required** : None

**Parameters** : None

## Success Response

**Code** : `200 OK`

**Content examples**


```json
{
    "success": true,
    "msg": [
        "cs2000",
        "cs2001",
        "cs2002"
    ]
}
```
