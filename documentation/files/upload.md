# Upload File

Used to upload file in to the system

**URL** : `/files/uploadFile`

**Method** : `POST`

**Auth required** : Yes

**Permissions required** : None

**Parameters** : projectId, file, uid

## Success Response

**Code** : `200 OK`

**Content examples**


```json
{
    "success": true,
    "msg": "File uploaded successfully"
}
```
