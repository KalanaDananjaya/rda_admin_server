# Get project FileIds

Used to get list of project file ids

**URL** : `/files/projectFiles`

**Method** : `GET`

**Auth required** : Yes

**Permissions required** : None

**Parameters** : projectId

## Success Response

**Code** : `200 OK`

**Content examples**


```json
{
    "success": true,
    "msg": [
        "0d6b2630-38db-11e9-afca-737741042d84",
        "33258140-38db-11e9-afca-737741042d84"
    ]
}
```
